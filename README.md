# LearnGit

git init - команда, що ініціалізує Git-репозиторій, тобто робить поточну папку новим git проєктом.
git add [name of a file] - команда, що додає файл в ініціалізований репозиторій. Файли, додані через git add знаходяться на 
низькому старті, очікуючи на commit. 
git add ./ - команда, що дозволяє додати всі файли в ініціалізований репозиторій.
git commit -m 'New commit' - дозволяє створити знімок (а не копію, це важливо!) проєкту на даній стадії його розвитку, яка 
називається коміт. В цьому коміті є лише зафксовані через git add зміни. Git фіксує їх і дає їмунікальний ідентифікатор, що 
дозволяє повернутися до неї в майбутньому. Флаг -m дозволяє дати коментар до коміта. Коментар - це лист в майбутнє, що 
дозволяє в майбутньому зрозуміти які саме зміни було внесено до репозиторію.
